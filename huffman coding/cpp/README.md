# Huffman Coding Compression Algorithm



## Usage



Compile the program
```
make main
```

Compression
```
./main <inputFile> -e <outputFile>
```
Example:
```
./main input.txt -e output.bin
```

Decompression
```
./main <inputFile> -d <outputfile>
```
Example:
```
./main input.bin -d output.txt
```

Clean up build files:
```
make clean
```
